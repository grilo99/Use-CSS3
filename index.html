Style HTML box properties

Every HTML element has box properties. These are the properties that control how the element is spaced on the page and control the position of the box contents. These can be set on any object or class in CSS. By default, an object will size itself to be able to display its contents. So, a div element with some text inside it will have a width and height that is sufficient to display the text.
